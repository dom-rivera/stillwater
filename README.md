# Stillwater 🌊

**A quiet place that never leaves your device.**

Stillwater is a free, private meditation app that runs entirely in your browser. No accounts, no tracking, no ads, no servers — just a single HTML file.

**Try it:** https://dom-rivera.github.io/stillwater/

## What's inside

- **Narrated tutorial** — a spoken walkthrough greets every visit
- **Guided sessions** — six original meditations, read aloud by your device's own calm voice, from a one-minute reset to an eight-minute sleep wind-down
- **Breathing exercises** — box breathing, 4·7·8, and coherent breathing, with spoken cues ("breathe in… hold… breathe out") synced to an orb that swells and settles with your breath
- **Quiet sit** — an unguided timer with a bell at each end and gentle spoken reminders to return to the breath (or pure silence, your choice)
- **Soundscapes** — rain, tide, and wind, synthesized live on your device; play them on their own or under any session
- **Settle-in countdown** — every session opens with ten seconds to arrive before the bell
- **Your practice** — streaks, minutes, session history, and mood check-ins

## Private by design

- Zero network requests after loading (the only external resource is fonts)
- History lives only in your own browser — it never touches a server
- Export your data as JSON, or erase everything, anytime, from the **You** tab
- Voice, bells, and soundscapes are all generated on-device with the Web Speech and Web Audio APIs — nothing is streamed or recorded

## Run it yourself

It's one file. Download `index.html` and open it in any modern browser — or host it anywhere static files are served.

For the best voice, use Edge on Windows, Safari on iPhone, or Chrome on Android. (In-app browsers inside chat apps often can't speak — the site will tell you if so.)
